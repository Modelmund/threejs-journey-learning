# debug ui
* 用于调试图形,即threejs官网examples中的调试面板
* 可以自行创建，也可以使用现有的库
* 库
  1. [dat.GUI](https://github.com/dataarts/dat.gui)
  2. control-panel
  3. ControlKit
  4. Guify
  5. Oui

# 创建debugui
1. 创建gui
2. 添加可配置项
3. 自定义参数
4. 自定义触发函数
